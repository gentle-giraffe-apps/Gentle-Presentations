# Resources

### Your Profile
- [Create a GitHub Profile README](https://youtu.be/ECuqb5Tv9qI) — video walkthrough of the special username repo, stats, badges, and GitHub Actions
- [awesome-github-profile-readme](https://github.com/abhisheknaiidu/awesome-github-profile-readme) — curated gallery of standout profile READMEs

### READMEs & Badges
- [Make a README](https://www.makeareadme.com/) — what to include, with an editable template and live preview
- [Shields.io](https://shields.io/) — generate SVG badges for build status, coverage, version, and more

### Commits & Coverage
- [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) — structured commit messages (`feat:`, `fix:`, etc.)
- [Codecov](https://about.codecov.io/) — coverage reporting that integrates with any CI; free for open source

### Discovery & Community
- [GitHub Trending](https://github.com/trending) — most-starred repos by day, week, or month
- [How to Contribute to Open Source](https://opensource.guide/how-to-contribute/) — GitHub's guide to finding projects and making your first contribution

- [Loom](https://www.loom.com/) — record screen + camera, share a link
